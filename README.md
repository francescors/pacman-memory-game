# Pacman Memory Game

Un jeu de memory sur le thème de Pac-Man, développé en Python 
avec la bibliothèque Turtle. Les objets et le décor sont entièrement 
dessinés par le programme.

## Aperçu

![Aperçu du jeu](screenshot.png)

## Fonctionnalités

- 4 niveaux de difficulté : Facile, Moyen, Difficile, Extrême
- 6 fruits à retrouver dessinés en pixel art (cerise, fraise, pomme, orange, melon, poire)
- Cases cachées par des fantômes animés
- Jauge de tentatives perdues
- Décor Pac-Man complet (personnages, points, fruits décoratifs)
- Option de rejouer en fin de partie

## Prérequis

- Python 3
- Bibliothèque standard uniquement (`turtle`, `random`, `time`)
- Polices à installer : `Orbitron` et `Rogue Hero 3D` (incluses dans le repo)

## Installation des polices

Avant de lancer le jeu, installe les deux polices :
- **Mac** : double-clic sur le fichier `.ttf` → "Installer la police"
- **Windows** : clic droit sur le fichier `.ttf` → "Installer"

## Lancement

```bash
python "script_principal.py"
```
